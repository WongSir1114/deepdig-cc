# 深度挖掘 · 商业版 v1.0.0

## 一、这是什么

深度挖掘是一个投研知识库系统。用户在 Obsidian 聊天面板中与 AI 对话，Claude Opus 引擎做深度分析，知识卡片自动写入 Obsidian Vault。

**不是 ChatGPT 套壳，不是量化平台，不是笔记软件。**

**是"以自生长为核心引擎的投研知识库系统"——加工深、说得懂。**

## 四、部署步骤

### A. 云服务器部署（已有阿里云·2核2G·Ubuntu·8.138.8.140）

```bash
# 1. 上传文件到服务器 /tmp/
#    （通过 Workbench 文件管理器）

# 2. SSH 到服务器
ssh admin@8.138.8.140

# 3. 复制部署文件
sudo cp /tmp/server.py /home/admin/deepdig/
sudo cp /tmp/user.html /home/admin/deepdig/frontend/

# 4. 重启服务
sudo systemctl restart deepdig

# 5. 验证
curl http://localhost:8000/api/health
# 预期：{"status":"ok","time":"...","users":0}
```

### B. 插件本地安装（用户）

```
1. 安装 Obsidian v1.5.0+
2. 安装 Node.js v18+
3. 终端执行：npm install -g @anthropic-ai/claude-code
4. 终端执行：claude login（浏览器登录 Anthropic）
5. 在 Obsidian 设置中启用"深度挖掘 · CC"插件
6. 打开插件（左侧 ribbon 图标）→ 注册/登录 → 开始使用
```

## 五、技术架构

```
用户本地                        云端（阿里云）
────────                        ────────
OB + CC 插件                    用户系统（注册/登录）
  ├─ 引擎卡解密注入             支付系统（微信/支付宝）
  ├─ 双调用模式                 订阅验证 API
  ├─ 知识底座扫描               套餐管理
  └─ 授权验证                   7天试用管理

AI 推理：本地 Claude Opus CLI
卡片存储：本地 Obsidian Vault
云端只做：用户管理+授权验证（不做 AI 推理·不存用户数据）
```
