### 2.2 技术实现

```
工程版 Vault：                            商业版插件 data.json：
                                          {
1-原子笔记/投资思想/                        "engine_cards": {
  巴菲特·安全边际原则.md                      "v": 1,
  巴菲特·经济护城河.md                        "k": "base64-aes-key",
  ... (22个文件)                             "d": "base64-aes-ciphertext"
                                           },
导出脚本 → 读内容 → JSON → AES-GCM →        "vault_template": { ... }
Base64 → data.json                        }

编译流程：
  main.ts → esbuild → main.js（含 data.json 内嵌的 engine_cards）
  
CC 调用时：
  ChatView.buildStdin() → decryptEngineCards() → 注入"=== 投资框架 ==="段落
  CC 看到完整的 22 张思想卡操作化规则 → 用框架判断 → 但不告诉用户引擎卡的存在
```

## 四、CC 插件授权集成

### 4.1 启动验证流程

```
插件 onload()
  ├─ ① 本地检查 JWT Token
  │   ├─ 存在且未过期 → 正常使用
  │   └─ 不存在或已过期 → ②
  │
  ├─ ② 调用 /api/verify-subscription
  │   ├─ 网络不可达 → 使用本地缓存的离线授权（7天宽限期）
  │   ├─ Token有效 + 套餐有效 → 缓存新Token → 正常使用
  │   ├─ Token有效 + 试用期内 → 显示"试用剩余X天" → 正常使用
  │   └─ Token有效 + 套餐过期 → 弹窗 → ④
  │
  ├─ ③ 正常使用 → 聊天面板就绪
  │
  └─ ④ 弹窗"您的套餐已过期"
      ├─ [续费] → 打开 deepdig.beaver-cloud.com/pricing → 支付
      ├─ [稍后] → 关闭弹窗·禁用发送按钮（只读已有卡片）
      └─ 支付完成 → 云端回调更新订阅 → 刷新 Token → 恢复使用
```

### 4.2 ChatView.ts 改动点

```typescript
// 新增属性
private authToken: string | null = null;
private subscriptionStatus: 'active' | 'trial' | 'expired' | 'unknown' = 'unknown';
private trialDaysLeft: number = 0;

// 启动时
async verifyLicense() {
    const token = await this.loadStoredToken();
    const resp = await fetch('https://deepdig.beaver-cloud.com/api/verify-subscription', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ token })
    });
    const data = await resp.json();
    this.subscriptionStatus = data.status;
    this.trialDaysLeft = data.trial_days_left || 0;
    
    if (data.status === 'expired') {
        this.showExpiredModal();
    }
    if (data.status === 'trial') {
        // 显示轻量提示——不阻断使用
        this.addMessage('system', `🎁 试用剩余 ${data.trial_days_left} 天`);
    }
}
```
