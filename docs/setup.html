⚡ 安装深度挖掘 · CC

Claude Opus 投研引擎 + Obsidian 知识底座。三步装好，开始深挖。

选择你的安装路径

路径 A：一键安装器 推荐

适合：不想碰终端、npm、环境变量。双击安装，全部自动完成。

1
下载安装器

🪟 下载 Windows 版   🍎 下载 macOS 版

约 60MB · 含 Node.js + CC CLI + 插件

2
双击运行安装器

自动完成：安装 Node.js → 安装 Claude Code CLI → 创建深度挖掘 Vault → 写入投研规则 → 装插件

3
打开 Obsidian → 开始使用

左侧 ribbon 图标 → 聊天面板。首次使用无需 License Key（7 天免费试用）。

路径 B：插件安装（你已有 Obsidian + Node.js）

1
安装插件

从 GitHub Releases 下载 main.js + manifest.json + styles.css，复制到 .obsidian/plugins/deepdig-cc/

2
初始化 Vault

运行初始化脚本（自动创建目录结构 + 写入投研规则）：

scripts\init-vault.bat "C:\Users\你的用户名\Documents\深度挖掘"
chmod +x scripts/init-vault.sh && ./scripts/init-vault.sh ~/Documents/深度挖掘
3
启用插件 → 开始使用

Obsidian 设置 → 第三方插件 → 启用"深度挖掘 · CC"。左侧 ribbon 图标 → 聊天面板。

路径 C：手动安装 高级

适合：开发者、已装好一切的用户。

1
安装 CC CLI 
npm install -g @anthropic-ai/claude-code
2
下载插件 + 初始化 Vault 
git clone https://github.com/deepdig-cc/deepdig-cc.git
cd deepdig-cc
./scripts/init-vault.sh ~/你的Vault路径
3
重启 Obsidian → 启用插件

🔑 配置 License Key

前 7 天免费——无需任何 Key，全部功能可用。

7 天后：deepdig.beaver-cloud.com → 订阅套餐 → 收到 Gumroad License Key → Obsidian 设置 → 深度挖掘 CC → 粘贴 Key → 验证。

💡 Key 验证一次即可，离线也能用，每 7 天在线确认一次订阅状态。

❓ 常见问题

一次深度分析多少钱？
约 $0.1-0.5，由 Anthropic 直接按量收取。API Key 是你自己的——我们不经手。
我的数据会上传到云端吗？
不会。所有分析在本地电脑完成，Vault 内容不出你的电脑。唯一外发请求是 License Key 验证（每 7 天一条 POST 到 api.gumroad.com）。
支持 macOS / Linux 吗？
支持。Windows 有一键安装器，macOS/Linux 用初始化脚本。
不付费能用吗？
首次安装 7 天免费试用，全功能。7 天后需要 License Key。
怎么获取 Anthropic API Key？
访问 console.anthropic.com → 注册 → API Keys → Create Key。粘贴到插件设置。
